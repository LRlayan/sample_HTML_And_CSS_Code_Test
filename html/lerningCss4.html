Learning Margin & Padding CSS

Margin Let's explain one and Padding with the help of the picture frame below.

As shown in the above image frame, the part indicated in orange color is the MARGIN and the part indicated in green color is known as the PADDING. These MARGIN and PADDING sections can be set to the required size using the required values when setting the CSS section.

Margin

Ex:

< body >

    < p class = " marg1 " > Margin Example < /p >

< /body >

Result:

Margin CSS

MARGIN can be written in CSS section in 04 ways. Those forms are below.

1ST Method

< head >
  < style >

    .marg1 {
        margin: px;
        background-color:;
        width:;
    }

  < /style >
< /head >

By setting the MARGIN in this way, the MARGIN will generally be ADDED around the area where the MARGIN is set.

2ND Method

< head >
  < style >

    .marg1 {
        margin: px px;
        background-color:;
        width:;
    }

  < /style >
< /head >

By setting the MARGIN in this way, only the TOP and BOTTOM MARGIN changes in the area where the MARGIN is set.

3RD Method

< head >
  < style >

    .marg1 {
        margin:px px px px;
        background-color:;
        width:;
    }

  < /style >
< /head >

By setting the MARGIN in this way, changes in the TOP, RIGHT, BOTTOM, LEFT MARGIN respectively will occur clockwise around the relevant part where the MARGIN is set.

4TH Method

< head >
  < style >

    .marg1 {
        margin-right:;
        background-color:;
        width:;
    }

  < /style >
< /head >

According to the MARGIN set above, a MARGIN can be added to only one SIDE. In this way, MARGIN can be added only to the required profile from all the four profiles.

Ex:

    margin-right
    margin-bottom
    margin-left
    margin-top

Padding

Ex:

    < p class = " padd1 " > Padding Example < /p >

Padding CSS

You can write the CSS part in PADDING in 04 ways. It is written in the same way as the 1st, 2nd, 3rd and 4th of the MARGIN explained above.

Ex:

< head >    
  < style >
    .padd1 {
        padding:;
        background-color:;
        width:;
    }
  < /style >
< /head >
