Learning Position CSS

There are 03 types of positions.

  • Fixed
  • Relative
  • Absolute

Fixed Position

  • Here we can use this FIXED POSITION if we want to keep only the required part of the WEB PAGE in a vertical position without moving it down or up.

Ex:

    < p class = " posf " Fixed Position css < /p >

Result:

Fixed Position Example



Below is the corresponding CSS section.

    .posf {
        position:fixed;
        top:630px;
        left:5px;
        background-color:yellow;
        width:250px;
        color:green;
        padding:8px;
    }

Relative Position

  • Here, when giving a RELATIVE POSITION, its POSITION is decided by the computer in comparison to what is set above the part that is expected to give the RELATIVE POSITION.

Ex:

    < p class = " posr " > Relative Position css < /p >

Result:

Relative Position Example


Below is the corresponding CSS section.

    .posr {
        position:relative;
        top:5px;
        left:5px;
        color:red;
    }

Absolute Position

  • When performing an ABSOLUTE POSITION, the computer obtains its POSITION relative to the TOP MARGIN and LEFT MARGIN of the WEB PAGE.

Ex:

    < p class = " posa " Absolute Position css < /p >

Result:

Absolute Position Example


The corresponding CSS section is shown below.

Ex:

    .posa {
        position:absolute;
        top:1240px;
        left:10px;
        color:red;
    }
