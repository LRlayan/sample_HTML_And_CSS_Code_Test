  • Here, after setting the LEFT FLOAT and the RIGHT FLOAT, the empty space left between the two FLOATs will be entered into that empty space. Otherwise, if you want the typed part to be DISPLAY below LEFT and RIGHT FLOAT, you should follow the HTML part and CSS part below.

Ex:

    < p class = " sec9 " Type Your Pharagraph < /p >

Ex:

    .sec9 {
        clear: both;
    }

  • In the above CSS clear:both; What is meant is that the LEFT and RIGHT FLOAT set above the paragraph typed below the LEFT and RIGHT FLOAT should be typed CLEAR. Then the TYPE section can be written as a single paragraph.


  • Here the CSS should be written in the section where LEFT RIGHT FLOAT should be removed. That is, to the paragraph typed below LEFT and RIGHT FLOAT. There, sec9 is the CLASS NAME related to that CSS.


  • If you want to remove only one of LEFT and RIGHT FLOAT, clear:left; and clear:right; Only the FLOAT you want to remove should be TYPED in the corresponding field
