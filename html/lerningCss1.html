Learning The In Line CSS & Internal CSS

         

  • When setting INTERNAL CSS sections it is Style Open Tag < Style > One and Style Close Tag < /Style > should be set between the two.

In line CSS

In Line CSS is the part of CSS included in the HTML code itself It should be done in the following way. Also, in order for the CSS prepared here to work, the STYLE TAG should be used in the following manner.

Ex:

    < p style = "color:red font-size:80%; " > In line CSS < /p >

Result:

In line CSS

Internal CSS

Internal CSS is to enter the CSS section in the HEAD TAG as follows according to the HTML code.

Ex:

< style >
    p {
      color:green;
      font-size:80%;
    }
< /style >

Here every CSS you set should be between STYLE OPEN TAG( < style > ) and STYLE CLOSE TAG ( < /style > ).

CSS can be typed anywhere in relation to HTML and as a standard it is included in the HEAD TAG.

Result:

Internal CSS

Learning The Class & ID

Class

A CLASS can be entered multiple times in the same PAGE, and several CLASSES can be added to the same HTML TAG. These methods can be explained by the following examples.

Ex 01:

    < p class = " Type Class Name " > Class Example < /p >

Result:

Class Example

The CSS related to this CLASS should be entered between HEAD OPEN TAG ( < head > ) and HEAD CLOSE TAG (< /head > ) as follows.

Ex:

    .css2 {
        color:blue;
        background-color:orange;
        padding:5px;
        width:105px;
    }

Ex 02:

    < p class = " Type 1st Class Name Type 2nd Class Name ; " Class Example < /p >

Result:

Class Example

Below is the section of CSS related to Ex 02.

Ex:

    .css3 {
        color:brown;
    }

    .css4 {
        font-size:150%;
    }

After using a CLASS for an HTML TAG in CSS, when creating the corresponding CSS section, dot   in front of the CLASS NAME used in the HTML TAG. Appending a  (.) is mandatory. When a PHARAGRAPH TAG has not been CLSSed, changes can be made to all PHARAGRAPH TAGs that have not been CLASSED at once. When writing a CSS to the PHARAGRAPH TAG that is not CLASSED No need to apply a dot on the front.

Ex:

    p {
      color:red;
      background-color:grey;
    }

ID

An ID can be used only once per PAGE.

Ex:

    < p ID = " css7 " > ID Example < /p >

Result:

ID Example

The CSS section related to this is shown below.

Ex:

    #css7{
            color : #E74C3C;
            font-size : 150%;
   }

When using an ID, it is mandatory to prefix the CSS NAME with a ( # ) symbol when writing the CSS section.
