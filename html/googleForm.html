How to create a link on the same web page

What is done here is to create a "Link" to look at an "image" at the top or anything else when you have moved to the bottom of the "web page". Or creating a "link" to go to the top of the "web page" when the "web page" has gone down.

Ex:
Let's examine below how to create a "Link" to automatically move to the "Google" "image" at the top of this "Web Page" when the word "Google Logo" is "Clicked" at the bottom of it.

Here first, an "ID" should be created for the code of the "Google" image above. This is done to identify where you should go when you click the word "Google Logo" created by you.

This ID can be created by "Opening" a "Paragraph Tag".( < P ID = "top" > < /p >)

Ex 01:
< p Id ="top" > < img src ="image link .jpg" > < /p >

      Here "Top" is the "ID Name". This code is related to the "Image" of "Google" above. This code should be "typed" where you want to replace the "Image".
      (Top of Web Page)

Second, the word "Google Logo" should be created and "Linked" to the "Google Image" above using the "ID" set.

Ex 02:
< a href = "#top" > Google Logo < /a >

     Here "#" should be added to "top".("#top")

Result:

Google logo
