How To Create Next Button

Ex:
<input type ="submit">

Result:

Here, remove the word "SUBMIT" and replace it with the word "NEXT" and follow the code Value ="Next".

Ex:
<input type ="submit" value ="Next">

Result:

How to "Link" another "Web Page" to the "Next Button" created above.

Ex:
<a href ="table.html"> <input type ="submit" value ="Next"> </a>

Here "table.html" is the name of the other "Web Page" you prepared. Its name should be "pasted" in the same way without making any changes. After that, the extension .html should be typed at the end of the name.
